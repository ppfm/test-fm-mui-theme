# Installation
```
npm i fabby-mui-theme
```
# Usage

Add 
```
import {ThemeProvider} from '@material-ui/core/styles';
import theme from 'fabby-mui-theme';
```

to App.js, then wrap your code
```
<div className="App">
    <ThemeProvider theme={theme}>
        <header className="App-header">
```

Finito!
